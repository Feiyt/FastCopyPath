# FastCopyPath

一个极速、轻量的 Windows 右键菜单增强工具，用于一键复制文件或文件夹路径。

## ⚖️ 实现原理对比

**本项目本质上是通过修改 Windows 注册表来添加右键菜单。**
根据注册表**调用目标**的不同，分为两种模式：

| 特性 | 🥇 模式 A：调用专用程序 (推荐) | 🥈 模式 B：直接调用系统命令 |
| :--- | :--- | :--- |
| **注册表行为** | 指向 `FastCopyPath.exe` | 指向 `cmd.exe` |
| **视觉体验** | **完全静默** (无窗口闪烁) | ⚠️ 有黑色 CMD 窗口一闪而过 |
| **文件依赖** | 需保留 `bin` 文件夹 | **无依赖** (设置完可删除项目) |
| **安装脚本** | `install.bat` | `reg_only.bat` |

## 📦 模式说明

### 1. 调用专用程序 (`install.bat`)
注册表会调用我们编写的极简 C++ 程序 (`FastCopyPath.exe`) 来执行复制操作。
*   **优势**：利用 C++ 程序处理剪贴板，通过 Windows API 隐藏窗口，实现**无黑框静默运行**。
*   **注意**：安装后不能删除 `bin` 文件夹，因为注册表需要找到这个程序。

### 2. 直接调用系统命令 (`reg_only.bat`)
注册表直接调用 Windows 自带的 `cmd` 和 `clip` 命令。
*   **优势**：**极致洁癖**。脚本修改完注册表后，您可以**删除整个项目文件夹**，功能依然有效（因为只依赖系统自带命令）。
*   **劣势**：受限于 `cmd` 机制，运行时会有瞬间的黑框闪烁。

## 🔧 其他工具
*   **`uninstall.bat`**: 一键卸载，清除所有注册表残留。
*   **`release.bat`**: 编译并生成 `dist` 发布包，方便分享给朋友（无需对方安装编译环境）。

## 💻 开发环境
*   语言: C++ (MinGW g++)
*   编码: UTF-8
*   依赖: 无 (静态编译，独立运行)
